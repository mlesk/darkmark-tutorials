0001 — Vectors: The Atom of AI Data

Linear Algebra Pillar · ~20 minutes

🧩 Puzzle Piece: In the transformer pipeline (Lesson 0000), the first step after tokenization is the embedding layer — converting each token into a vector. Every word, every image patch, every piece of data flowing through a transformer is a vector. This lesson builds the foundation: what a vector is, and what you can do with one.
Every word you type into ChatGPT, every image Midjourney generates, every song a recommendation system suggests — all of them start as vectors. A vector is the fundamental unit of data in AI. If you understand vectors deeply, you understand the alphabet that every AI system reads.

What Is a Vector?

There are two ways to see a vector, and you need both:

The Geometric View

A vector is an arrow in space. It has:

  • Direction — where it points
  • Magnitude (or length) — how far it reaches

In 2D, a vector v = (3, 1) means "go 3 steps right, 1 step up." Draw it from the origin — it's an arrow pointing to the point (3, 1).

The Algebraic View

A vector is a list of numbers:

v = [3, 1]
2 dimensions
w = [0.2, −0.8, 0.5, 0.1]
4 dimensions
e = [0.12, −0.03, …, 0.41]
768 dimensions

The number of entries is the vector's dimension. A vector in 2D lives on a flat plane. A vector in 768D lives in a space you cannot visualize — but the same rules apply. This is the core trick of AI: math that works in 2D works in 768D exactly the same way.

Key insight: The geometric view builds intuition. The algebraic view scales to any number of dimensions. AI systems use vectors with hundreds or thousands of dimensions — but the operations are identical to what you do in 2D. If you can reason about arrows on a plane, you can reason about word embeddings, image features, and user preferences.

Three Operations That Matter

1. Addition: v + w

Geometrically: place the tail of w at the tip of v. The result is the arrow from origin to the new tip. Algebraically: add component by component.

[3, 1] + [1, 2] = [4, 3]

In AI terms: if one vector represents "king" and another represents a "gender direction," adding them might approximate "queen." This is not a metaphor — word embeddings literally work this way.

2. Scalar Multiplication: c · v

Geometrically: stretch or shrink the arrow by factor c. If c is negative, flip the direction.

2 · [3, 1] = [6, 2] (stretch)
−1 · [3, 1] = [−3, −1] (flip)

In AI terms: scaling changes the "strength" or "intensity" of whatever the vector represents.

3. Dot Product: v · w

Geometrically: measures how much two vectors point in the same direction. Positive = similar direction, zero = perpendicular, negative = opposite directions.

[3, 1] · [1, 2] = 3×1 + 1×2 = 5

In AI terms: the dot product is the fundamental measure of similarity. When an AI system tells you two things are "similar," it computed a dot product (or its normalized cousin, cosine similarity). This is how recommendation systems find songs you'll like, how search engines rank documents, and how attention mechanisms in transformers decide which words are relevant to each other.

The dot product is arguably the single most important operation in all of applied AI. It appears in: similarity search, attention mechanisms, recommendation systems, classification, clustering — essentially anywhere an AI system needs to compare two things.

Vectors in AI: Three Real Examples

Domain What Gets Vectorized Dimension What the Dot Product Measures
LLMs (GPT, Claude) Words / tokens 768–12,288 Semantic similarity — "king" is close to "queen," far from "bicycle"
Image Generation Image patches Typically 512–1,024 Visual similarity — which patches of an image relate to each other
Recommendation Users & items 50–200 Preference match — how much a user's taste vector aligns with an item's feature vector

In every case, the AI system is doing the same thing: representing things as vectors, then comparing them with dot products. The magic is in how the vectors are constructed — that's where learning (training) comes in — but the operations are just the three you learned above.
